Help for budget calculator:

Base Conversion Rate:

The number of conversions divided by the total number of impressions.

CPM:

Cost per mille (1000 impressions):

Example: If we get 5000 impressions after spending 231 $, then the CPM will be 5000/231=21.6 $

Sample size:

The number of required impressions in the test variant.

Budget:

How much budget to spend on the test variant.

Budget over 3 weeks:

How much budget to spend each week on the test variant.
