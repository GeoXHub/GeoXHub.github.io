Glossary

Base Conversion Rate:

The number of conversions divided by the total number of impressions.

Example: Lets say you have a campaign you want to test and the number of clicks is 20.000 and the number of impressions is 300. Then the base conversion rate will be:
Clicks / Impressions
=
300 / 20.000
= 0.015 = 1.5%

CPM:

Cost per mille (1000 impressions).

Example: If we get 12.000 impressions after spending 231 $, then the CPM can be calculated as:

Cost / Impressions
* 1000 =
231 $ / 12.000
* 1000 = 19.25 $

Frequency:

The frequency of which you target consumers

Example: If you have 20.000 impressions split out on 13.000 consumers then your frequency will be:

Impressions / consumers
=
20.000 / 13.000
= 1.54
