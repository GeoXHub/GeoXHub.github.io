Glossary

5 steps for successful A/B Testing

  1. Determine test variable: What tactic are you testing? Examples: Call to action, ad imagery, targeting, landing page

  2. Identify test KPI: What is the primary metric that will determine the winning tactic? Examples: CTR, CPSV, ROAS

  3. Define test and control groups. Control group reflects the standard/usual tactic used. Test group reflects the altered tactic that you want to compare to your standard approach. Remember that the test variable is the only difference between both groups. All other variables such as budget, frequency, creative and duration must be kept the same.

  4. Split your sample groups equally and randomly. There must be no overlap in users between the sample groups. Most platforms offer A/B tools that automatically create randomized delivery groups.

  5. Determine sample size. Use Budget Calculator to determine minimum sample size and budget.

Base Performance Rate:

The number of performance metrics (i.e. test KPI) divided by the total number of fx. impressions. Depending on the test KPI, this could be clicks/visits/sales. Look at historical data to get a base performance rate that you expect the Control group to perform at.

Example: Lets say your test KPI is Clicks, and you want to test if one campaign is better than another.
Using historical data we can see that we usually get 20,000 impressions and 300 clicks.
Now we can caluclate the base performance rate to be:
Clicks / Impressions
=
300 / 20,000
= 0.015 = 1.5%

Example: Lets say your test KPI is conversions and you want to test if one visitation stategy leads to more conversions than another visitation strategy.
Using historical data we see that we usually get 500 visits and 15 conversions. Now the base performance rate can be calculated by:
Conversions / Visits
=
15 / 500
= 0.03 = 3 %

Buying CPM:

Cost per mille (1,000 impressions).

Example: If we get 12,000 impressions after spending 231 $, then the CPM can be calculated as:

Cost / Impressions
× 1,000 =
231 $ / 12,000
× 1,000 = 19.25 $

Avg. campaign Frequency:

The frequency of which you target consumers

Example: if campaign runs for 4 weeks and frequency is 1 pr. week, the avg. campaign frequency is 4 × 1 = 4
